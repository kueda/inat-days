iNat Streaks

Some iNat users keep it regular. Like, really regular. These charts show content created over time by users with the most unique days for each content type.

  • Observation streaks by date observed
  • Observation streaks by date added
  • Identification streaks by date added
Some stuff to keep in mind
  • each visualization can be zoomed in via mouse pad or scroll wheel
  • individual dots represent days
  • observation counts are only using observations with photos
  • the rows of each visualization are sorted by number of days for each user, so for observations by date added, the user at the top will be the user who has added observations to the site on the most unique days.
